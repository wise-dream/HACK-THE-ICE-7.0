# Поддержка++ — обзор фронтенда

## Цель фронтенда

Сделать максимально простой и лёгкий интерфейс, который:
- работает на слабых смартфонах и 3G
- понятен людям без цифровых навыков
- даёт персональный список льгот и скидок
- позволяет быстро найти «свою» льготу и понять, что делать дальше

Технически: Nuxt 3/4 + PWA + Tailwind + Pinia + FSD.

## Основные роли

1. **Льготник (основная роль)**
   - пенсионер
   - человек с инвалидностью
   - многодетный родитель
   - малоимущий и т.д.
   - Цель: быстро увидеть, какие льготы ему доступны, где/как получить, что скоро закончится.

2. **Помогающий родственник / соцработник**
   - Пользуется приложением за другого человека.
   - Цель: собрать список актуальных льгот, распечатать/отправить.

3. **Администратор/редактор** (для хакатона можно скрытый простой интерфейс)
   - добавляет/правит льготы (JSON-база).
   - В MVP можно ограничиться json-файлом + mock-API, но UI под это можно наметить.

## Основные пользовательские сценарии

1. Первый заход:
   - Попадает на лендинг («что это за сервис»).
   - Нажимает «Войти/Зарегистрироваться».
   - Проходит простую регистрацию (email + sms/mock).
   - Выбирает категорию льготника, регион, интересующие категории льгот.
   - Сразу попадает на экран «Мои льготы».

2. Ежедневное использование:
   - Открывает приложение с иконки (PWA).
   - Видит карточки актуальных льгот и акций.
   - Фильтрует по типу («лекарства», «транспорт» и т.д.).
   - Открывает карточку, смотрит «что нужно», «куда идти».

3. Подготовка визита в МФЦ/поликлинику:
   - Находит нужные льготы.
   - Нажимает «Собрать список».
   - Экспортирует в PDF/печать.

4. Работа с сетью:
   - При слабом интернете видит skeleton-лоадеры.
   - При отсутствии сети - простой оффлайн-экран с последним кешированным списком льгот (если есть).

## Навигационная модель

- **Mobile-first, одна основная вкладка «Мои льготы»**
- Нижняя навигация (bottom bar, 3–4 пункта):
  - «Мои льготы»
  - «Каталог» (поиск и фильтры)
  - «Избранное» (опционально)
  - «Профиль»
- На десктопе — слева вертикальное меню, справа контент.
